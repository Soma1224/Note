# unity的UI框架

- 用Json信息保存所有面板（界面），以及所有页面的路径



**UI面板的状态**

- prefab
- 显示
- 不显示



页面的显示和移除适合用栈来做



如何在点击主菜单的按钮，弹出所选框的时候不能再操作主菜单？

为主菜单添加生命周期。

生命周期有：

- 进入显示
- 暂停
- 继续运行状态
- 移出显示
